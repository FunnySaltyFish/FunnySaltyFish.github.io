Python 3.10正式发布，你尝鲜了吗？

本文参考自 Python官方文档 ： Python Release Python 3.10.0 | Python.org

在正值国庆假期人山人海的2021年10月4号，Python官方正式发布了Python3.10.0。作为一只假期期间宅着不动的coding人，自然是第一时间体验了一波。相较于之前的版本，该版本有以下主要变更。
